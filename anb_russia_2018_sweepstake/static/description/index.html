Analytic Shortcut Button In Project

Analytic account shortcut button available in projects form.

Translated to:

Description

Obtaining analytic account with one click

This module adds a new button in projects form, which links to the analytic account of the opened project. A shortcut to get the analytic account of a project with only one click.

Installation

To install this module, you need to:
  • You must have installed or available to install the module analytic.
  • You must have installed or available to install the module project.

Usage

To use this module, you need to:

First, check that the user who is going to use this functionality has Analytic Accounting access rights. To do that, go to Settings > Users > Users, select the user and look for the mentioned right under the Technical Settings section.

After setting that configuration:

  • Go to Project > Dashboard to see the kanban view of projects.
  • Choose a project, click on its More button and then on Settings to open the project form view.
  • Click on the new button Analytic accounts at the top right corner of the form.
